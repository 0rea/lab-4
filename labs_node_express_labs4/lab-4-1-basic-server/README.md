# Lab 4.1 - Basic Node.js & Express Server

## วิธีรัน
1. ติดตั้ง dependencies:
   npm install

2. รัน HTTP server:
   npm run start:http

3. รัน Express server:
   npm run start:express

## Endpoints
- GET / (root)
- GET /students
- GET /students/:id
- GET /students/major/:major
- GET /stats (Express)
